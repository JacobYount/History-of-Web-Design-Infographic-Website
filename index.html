The History of Websites

By: Jacob Yount

1991 - 2024

What is HTML?

HTML stands for Hypertext Markup Language. It's the standard markup language for creating and designing web pages. HTML is used to structure the content of a web page by using various tags and elements to define different parts of the page such as headings, paragraphs, lists, links, images, and more. It provides the basic building blocks for creating web content and works in conjunction with other languages like CSS and JavaScript to create visually appealing and interactive web pages. HTML documents are interpreted by web browsers, which render them into the visual web pages that users see and interact with.

CSS

CSS (Cascading Style Sheets) is a style sheet language used to describe the presentation of a document written in HTML. CSS describes how elements in a web page should be displayed. CSS makes it easy to control the layout, appearance, and formatting of HTML elements. With CSS, you can specify properties such as colors, fonts, margins, padding, borders, and positioning for various elements on a webpage.

  • Explore CSS

Javascript

JavaScript is used for creating interactive and dynamic content on websites. It is commonly employed to add functionality such as user interaction, form validation, animations, and dynamic updates to web pages. JavaScript runs on the client side, meaning it executes within the user's web browser, enabling real-time interaction without needing to communicate with the server.

Birth of the Web

The first ever web page was was published on August 6th 1991 by Tim Berners-Lee using HTML 1.0, it included information on the World Wide Web and outlined formats, protocols, and how you could create web pages.

Early web pages were mainly text-based because internet connections were slow and technology was limited. No special design or page structure was present, but the basic HTML tags for paragraphs and links made their debut.

The Mid-90s

In the mid 90s the concept of web design became more of a mainstream idea and the concept of GUIs was introduced with colors, table-based layouts, flashy GIFs, and website counters. Although websites later moved away from table-based design, the idea of structure continued.

Additionally, HTML 2.0 was made in 1995, it including all of the features of HTML 1.0 plus some new concepts like structural elements for paragraphs, headings, ordered and unordered lists

The Late 90s

In the late 90s Flash was created; Flash is a software platform that allowed web designers to incorporate music, video, and animation into their websites, making for a more dynamic and interactive audio-visual experience. Flash introduce several new concepts to web design including interactive menus, 3D buttons, animation, and colour-changing navigation. Overall, the introduction of colors, images, animations, etc. made for a more fun user experience when browsing the web and made web design more immersive.

Additionally, HTML 3.0 was invented in 1997 supporting all of HTML 2.0 features and new concepts such as tables, text flow around figures, interactive forms, text formatting, and enhanced image features giving developers better control over image size, alignment, and text descriptions.

1999 to Early 2000s

In 1999 HTML 4.0 was introduced and became hugely popular due to how versatile it was. HTML 4.0 supported CSS (Cascading Style Sheets), allowing developers to separate the structure of a document from its presentation and let developers define more particular styles – such as larger font sizes for sub-headings – across multiple pages of a single website without having to code each element individually.

Additionally, HTML 4.0 supported scripting languages like Javascript and PHP allowing for scripts to be embedded into websites making them much more interactive and versatile.

Late 2000s to Early 2010s

The concept of responsive web design is invented by Ethan Marcotte in response to the popularization of smart phones and other multimedia devices. The idea behind responsive design was that a single website could respond and adapt to different display environments, such as a phone or a laptop. Giving users the same experience on their mobile device as on their desktop computer, meaning increased efficiency in web development and maintenance.

Additionally, the concept of responsive web design led to a web design trend called flat design. This trend embraced an efficient and visually appealing minimalist two-dimensional style emphasizing functionality over ornamental design elements.

2014

In 2014 HTML 5.0 was released and has been used worldwide ever since. HTML 5.0 included all of the features of the previous versions of HTML plus it allowed for much more versatility including many new features.

New Form elements were added such as Email and Password, Email checks if the user has input a valid email address and the password tag is designed to capture passwords securely.

Audio tags were also introduced allowing developers to embed and integrate audio content directly and seamlessly into web pages.

Semantic/Structural tags such as the header and footer tags were also introduced providing more organization and structure to HTML pages. These tags provided a clearer hierarchy and meaning to different sections of a webpage.

Section tags were introduced defining a distinct section within an HTML page. This helps in organizing and delineating content throughout a webpage.
